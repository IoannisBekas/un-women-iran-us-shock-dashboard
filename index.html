How to read this dashboard

Screening tool, not a causal estimate

The score ranks exposure to plausible escalation pathways. It does not estimate how many women or girls became food insecure after the shock, because the current public data pull does not include sex- and age-disaggregated food-security outcomes.

Key terms and model components

Shock Exposure Index

A 0-100 screening score built from nine bounded 0-1 components and documented weights.

IPC Phase 3+

Population in crisis or worse food insecurity. IPC is total-population data in this workbook, not sex-disaggregated.

Gender proxy priority

A fallback priority score using structural and proxy gender indicators when measured women/girls outcomes are unavailable.

Scenario sensitivity

Estimated score movement under oil, fertilizer, food-price and assistance-cut stress tests.

Data readiness

A check on whether a country has enough pre/post and outcome data for more formal causal work.

Additional public layers

IDMC, IOM DTM and WFP outcome layers are now formally weighted in the refreshed score, with missing values flagged.

    Method boundary

    Method and interpretation boundaries

    This dashboard summarizes the official model workbook as a screening and monitoring tool. It helps identify countries that may need closer attention under regional escalation scenarios by combining food-price pressure, import and fertilizer exposure, acute food insecurity, displacement, assistance pressure and gender vulnerability. The index is expert-weighted, documented and sensitivity-tested using OECD/JRC composite-indicator guidance. It is not a causal estimate and should not be used to state that the escalation caused a specific increase in food insecurity among women or girls.

    Rationale for the weights

    The weights translate the project theory of shock transmission into a transparent prioritization score. They are documented expert judgment for this UN Women screening exercise, not universal or externally mandated weights.

    • Pathway coverage: the score covers the main escalation pathways: prices, import exposure, fertilizer/agriculture, acute food insecurity, displacement, assistance pressure and gender vulnerability.
    • Severity anchor: IPC acute food insecurity receives the largest single weight, 17%, because current crisis-or-worse severity is the strongest prioritization signal.
    • UN Women lens: gender vulnerability receives 15%, equal to food-price pressure, so the index remains gender-sensitive rather than becoming a generic macro or food-price model.
    • Data quality discipline: public WFP outcomes receive 5% because coverage is sparse; the older displacement proxy is kept at 7% and complemented by an 8% public IDMC/IOM layer to reduce double counting.
    • Balanced upstream risks: import exposure and agriculture/fertilizer exposure each receive 12% because they are important transmission pathways but less direct than observed food-insecurity severity.
    • External references are used carefully: IMF-style shock logic informs pathway selection, while OECD/JRC guidance informs composite-index construction. No external weight set is copied.
    • Robustness testing: alternative weighting schemes are included in the Weight_Sensitivity output, so decision makers can see whether priorities change under price/import-heavy, gender/humanitarian-heavy or displacement/outcome-heavy assumptions.
