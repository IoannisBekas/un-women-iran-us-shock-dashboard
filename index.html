How to read this dashboard

Prioritization, not causal attribution

The score ranks exposure to plausible escalation pathways. It does not estimate how many women or girls became food insecure after the shock, because the current public data pull does not include sex- and age-disaggregated food-security outcomes.

Key terms and model components

Shock Exposure Index

A 0-100 screening score built from nine bounded 0-1 components and documented weights.

IPC Phase 3+

Population in crisis or worse food insecurity. IPC is total-population data in this workbook, not sex-disaggregated.

Gender proxy priority

A fallback priority score using structural and proxy gender indicators when measured women/girls outcomes are unavailable.

Scenario sensitivity

Estimated score movement under oil, fertilizer, food-price and assistance-cut stress tests.

Data readiness

A check on whether a country has enough pre/post and outcome data for more formal causal work.

Additional public layers

IDMC, IOM DTM and WFP outcome layers are now formally weighted in the refreshed score, with missing values flagged.

    Method boundary

    How the dashboard should be cited internally

    This dashboard communicates the official model workbook as a screening and monitoring tool. The model uses expert-assigned composite-index weights, OECD/JRC-guided documentation, scenario sensitivity and a causal-readiness layer. The weights are not IMF weights; IMF-style shock logic is used only to define food, fuel, fertilizer and import-exposure pathways. It should not be used to claim that the escalation caused a specific increase in food insecurity among women or girls.

    Why these weights are defensible

    The weights are not presented as universal, causal or externally mandated weights. They are documented expert judgment for a UN Women screening index, aligned with OECD/JRC composite-indicator practice.

    • Theory first: each weight maps to a plausible shock pathway: prices, import exposure, fertilizer/agriculture, acute food insecurity, displacement, assistance pressure and gender vulnerability.
    • Outcome severity anchors the score: IPC acute food insecurity receives the largest single weight, 17%, because current crisis-or-worse severity is the strongest prioritization signal.
    • UN Women mandate is visible: gender vulnerability receives 15%, equal to food-price pressure, so the index does not become a generic macro or food-price model.
    • Data quality affects weight size: public WFP outcomes receive 5% because coverage is sparse; the older displacement proxy is kept at 7% and complemented by an 8% public IDMC/IOM layer to reduce double counting.
    • Upstream risks are material but moderated: import exposure and agriculture/fertilizer exposure each receive 12% because they are important transmission pathways but less direct than observed food-insecurity severity.
    • Robustness is tested: alternative weighting schemes are included in the Weight_Sensitivity output, so decision makers can see whether priorities change under price/import-heavy, gender/humanitarian-heavy or displacement/outcome-heavy assumptions.
