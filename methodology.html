Methodology

The dashboard is a screening and monitoring tool for the Iran-USA regional escalation food-security story. It combines price, import, agriculture, food-insecurity, displacement, assistance and gender-vulnerability pathways from documented public-source indicators.

Analytical boundary

What the dashboard can and cannot say

The dashboard is a screening and monitoring tool. It ranks country exposure to plausible Iran-USA regional escalation pathways and helps prioritize where closer food-security and gender-sensitive monitoring may be needed. It does not estimate how many women or girls became food insecure, because public sex- and age-disaggregated food-security outcome data are not available across the country set.

The before/after module is descriptive price evidence. It compares matched food-price observations after the shock window with the same calendar month one year earlier. It does not prove that the escalation caused the observed price movement. The dashboard should therefore be read as evidence for prioritization, monitoring and data-readiness decisions, not as a causal impact model.

Purpose

Screening model for regional escalation exposure

The dashboard summarizes a documented public-source indicator set and analytical calculations. The shock start date used for the pre/post checks is 2026-02-28. The model is organized around four linked analytical layers.

First, the Shock Exposure Index ranks each country on a 0-100 screening scale. Higher scores mean higher relative exposure and monitoring priority, not measured harm. Second, scenario sensitivity stress-tests how country scores move under oil-price, fertilizer, food-price and assistance-cut assumptions. Third, the before/after price module compares available food-price series across matched months. Fourth, readiness and caveat fields explain where the available evidence can support stronger interpretation and where it cannot.

The exposure framework combines domestic food-price pressure, food and fuel import exposure, agriculture and fertilizer exposure, IPC acute food insecurity, public WFP food-security outcome availability, a legacy external displacement proxy, an IDMC/IOM internal-displacement layer, assistance and funding pressure, and gender vulnerability. These pathways are intentionally broader than prices because the food-security story is also shaped by import dependence, assistance access, displacement and structural vulnerability.

Visual interpretation

How to read the chart explanations

Each dashboard visual now includes a compact explanation block with three parts: how the visual is calculated, how to read the result, and which dashboard fields feed the chart or table. These blocks are intended to make the dashboard auditable during meetings without requiring users to inspect the workbook.

For composite scores, the dashboard shows decomposition tables. In the Shock Exposure Index table, each row is calculated as component_score x component_weight x 100. In the gender proxy table, each row is calculated as the visible input score multiplied by its gender-proxy coefficient, with 0-1 inputs converted to contribution points. The contribution rows show arithmetic drivers of the score; they do not identify causal channels or measured impacts.

The explanation blocks do not change the model weights, source data or published country scores. They are an interpretation layer on top of the current dashboard dataset.

Calculation

Shock Exposure Index

Each country receives a bounded 0-1 score for each pathway. Source indicators are normalized indicators so they can be compared on a common scale. Where a source value is missing, the component is set to a neutral value where needed and the missingness is flagged through the missing-component and data-confidence fields. The published index is calculated as:

Shock Exposure Index = 100 x sum(component_score x component_weight)

This formula creates a composite screening score, not a probability of crisis and not a measured outcome. A country can rank highly because several pressure pathways are elevated, or because one high-weight pathway is very strong. Alternative weighting schemes are provided so users can test whether the ranking is sensitive to the documented weights.

    Gender and displacement

    How displacement, IPC and gender proxy inputs are used

    Displacement is intentionally split into two model fields. The legacy external displacement proxy is retained for continuity with the earlier dashboard version and can be zero for countries where that older proxy does not capture the relevant movement pattern. The IDMC/IOM internal-displacement layer is the separate current input that uses public IDMC and IOM DTM evidence for internal displacement stocks, selected events and DTM summaries where available.

    This means a country such as Palestine or Yemen can show a zero legacy displacement proxy while still carrying nonzero internal-displacement pressure through the IDMC/IOM layer. The zero should therefore be read as "not captured by the legacy proxy", not as "no internal displacement pressure".

    IPC data are used in the IPC acute food insecurity component, but the public IPC pull is total population. The model does not convert total IPC values into women/girls counts. That conversion would require current sex- and age-disaggregated food security outcomes such as FCS, rCSI, LCS, assistance reach, access constraints, or female-headed household outcomes.

    The gender proxy priority score combines the published exposure score with structural gender vulnerability and selected humanitarian pressure inputs, including internal displacement and assistance pressure. The rationale is that displacement, aid access, funding cuts, food prices and labour-market vulnerability can have differential gender effects. The score is a prioritization lens for monitoring and field validation, not a measured impact estimate.

    Before vs after

    Food-price comparison method

    The before/after module uses the WFP/HDX food-price comparison input. The post-shock window starts in March 2026, the first full month after the shock start date. For each post-shock month in 2026, the comparison month is the same calendar month in 2025. March 2026 is compared with March 2025, April 2026 with April 2025, and so on.

    The comparison is made within the same country, market, commodity and unit. Prices are matched by the source-series identifier so that a post-shock observation is compared with the same market-commodity-unit series from the prior year. For each matched month, the country value is the median of (post_price / prior_year_price - 1) x 100 across the matched series.

    The overview keeps the highest positive matched same-month increase per country so that the overview module highlights where prices rose most. Countries with flat, negative or unmatched evidence remain visible in the Country Lens table because those cases matter for interpretation. A positive peak increase means food prices rose in matched WFP/HDX series after the shock window; it does not prove that the escalation caused the increase. Other shocks, seasonality, exchange-rate movements, conflict, market access and assistance changes may occur at the same time.

    Scenarios

    Stress tests, not forecasts

    Scenario values come from documented scenario sensitivity calculations. The Country Lens page groups the scenarios into a stress lens and a three-option shock level so the controls remain readable. Scenario deltas are point movements from the baseline score under a stated assumption. They are stress tests for planning discussion, not forecasts of what will happen.

    Oil-price scenarios emphasize import exposure, price pressure and assistance pressure. Fertilizer scenarios emphasize agriculture exposure, import exposure and acute food insecurity. Food-price scenarios emphasize domestic price pressure, acute food insecurity and gender vulnerability. Assistance-cut scenarios emphasize assistance pressure, acute food insecurity and displacement. Combined scenarios add the corresponding pathway shocks together under low, base and high assumptions.

    Caveats

    Main limitations

    The strongest communication line is to describe exposure, evidence strength and monitoring priority. The dashboard should not be used to claim that the escalation caused a specific food-security outcome, and it should not be used to estimate the number of women or girls who became food insecure. IPC is used as total-population acute food-insecurity severity; public IPC outputs used here are not sex-disaggregated.

    Results should therefore be interpreted as country exposure, evidence strength and monitoring priority across plausible escalation pathways. Before/after prices describe matched food-price changes after the shock window. Gender vulnerability is a structural and proxy prioritization lens. Scenario outputs are stress tests for planning discussion. The dashboard does not prove causality, estimate women/girls food-security impact, treat scenario deltas as forecasts, or treat negative, flat or missing price comparisons as evidence that a country has no risk.

    Traceability

    Public sources and calculation trace

    Dashboard values are read from documented public-source inputs and calculation checks. The main trace points are: final country ranking and score; bounded component scores; component weights and weighted contributions; WFP/HDX food-price evidence; latest national IPC timing and severity; scenario score movements; and price event-study, food-security outcome and gender-causal readiness flags.

    Supplemental public-source checks provide context and data-gap evidence from ACLED, CPI, food incidents, PortWatch and Findex. These layers help explain data availability and context. They do not replace the published Shock Exposure Index.
