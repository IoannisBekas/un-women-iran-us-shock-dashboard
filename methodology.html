Methodology

The dashboard is a screening and monitoring tool for the Iran-USA regional escalation food-security risk context. It combines price, import, agriculture, food-insecurity, displacement, assistance and gender-vulnerability components from documented public-source indicators.

Analytical boundary

What the dashboard can and cannot say

The dashboard is a screening and monitoring tool. It ranks country exposure in the Iran-USA regional escalation risk context and helps prioritize where closer food-security and gender-sensitive monitoring may be needed. It does not estimate how many women or girls became food insecure, because public sex- and age-disaggregated food-security outcome data are not available across the country set.

The before/after module is descriptive price evidence. It compares matched food-price observations after the shock window with the same calendar month one year earlier. It does not prove that the escalation caused the observed price movement. The dashboard should therefore be read as evidence for prioritization, monitoring and readiness checks, not as a causal impact model.

Purpose

Screening model for regional escalation exposure

The dashboard summarizes a documented public-source indicator set and analytical calculations. The shock start date used for the pre/post checks is 2026-02-28. The model is organized around four linked analytical layers.

First, the Shock Exposure Index ranks each country on a 0-100 screening scale. Higher scores mean higher relative exposure and monitoring priority, not measured harm. Second, the overview keeps cross-country readouts such as prices, rankings, regional averages, female informal, vulnerable and agricultural employment, and all-country displacement evidence, while the Country Lens page holds selected-country raw evidence. Third, the before/after price module compares available food-price series across matched months. Fourth, readiness and caveat fields explain where the available evidence can support stronger interpretation and where it cannot.

The exposure framework combines domestic food-price pressure, food and fuel import exposure, agriculture and fertilizer exposure, IPC acute food insecurity, public WFP food-security outcome availability, a legacy external displacement proxy, an IDMC/IOM internal-displacement layer, assistance and funding pressure, and gender vulnerability. These components are intentionally broader than prices because food-security risk is also shaped by import dependence, assistance access, displacement and structural vulnerability.

Visual interpretation

How to read the chart explanations

The user-facing dashboard now presents only one composite score: the official Shock Exposure Index. Other panels use raw or near-raw fields with units and source notes so readers do not need to interpret multiple score systems.

The overview page is meant for regional and cross-country interpretation. Selected-country evidence tables, raw gender/labour/financial-access fields and detailed country checks are kept in the Country Lens page, where the country selector gives those values context.

The only visible decomposition table is for the Shock Exposure Index. Each row is calculated as component_score x component_weight x 100. The contribution rows explain the arithmetic behind the official score; they do not identify causal channels or measured impacts.

The explanation blocks do not change the model weights, source data or published country scores. They are an interpretation layer on top of the current dashboard dataset. Terms that appear only in hidden workbook sheets or generated data traces are kept only for reproducibility and are not separate dashboard findings.

Calculation

Shock Exposure Index

Each country receives a bounded 0-1 score for each model component. Source indicators are normalized to a common scale before weighting. Where a source value is missing, the component is set to a neutral value where needed and the missingness is flagged through the missing-component and data-confidence fields. The published index is calculated as:

Shock Exposure Index = 100 x sum(component_score x component_weight)

This formula creates a composite screening score, not a probability of crisis and not a measured outcome. A country can rank highly because several pressure components are elevated, or because one high-weight component is very strong. Alternative weighting schemes are provided only so the internal model can be audited; those alternatives are not presented as user-facing dashboard scores.

    Gender and displacement

    How displacement, IPC and gender indicators are used

    Displacement is intentionally split into two model fields. The legacy external displacement proxy is retained for continuity with the earlier dashboard version and can be zero for countries where that older proxy does not capture the relevant movement pattern. The IDMC/IOM internal-displacement layer is the separate current input that uses public IDMC and IOM DTM evidence for internal displacement stocks, selected events and DTM summaries where available.

    In the simplified dashboard, readers see the public IDMC/IOM displacement evidence directly in an all-country table instead of comparing displacement proxy scores. This helps avoid the mistaken reading that an older zero proxy means no internal displacement.

    IPC data are used in the IPC acute food insecurity component, but the public IPC pull is total population. The model does not convert total IPC values into women/girls counts. That conversion would require current sex- and age-disaggregated food security outcomes such as FCS, rCSI, LCS, assistance reach, access constraints, or female-headed household outcomes.

    Gender vulnerability remains one component inside the official Shock Exposure Index. The user-facing dashboard no longer presents a separate gender-proxy priority score. If that field appears in hidden workbook or generated-data traces, it is retained only for reproducibility. Instead, the dashboard shows interpretable gender, labour and financial-access indicators directly: the overview uses cross-country gender/labour readouts such as female informal, vulnerable and agricultural employment, while the Country Lens page shows selected-country labour and Findex indicators such as vulnerable employment, agricultural employment, labour-force participation, account ownership, mobile money, saving, wage receipt and government payment receipt where available.

    Before vs after

    Food-price comparison method

    The before/after module uses the WFP/HDX food-price comparison input. The post-shock window starts in March 2026, the first full month after the shock start date. For each post-shock month in 2026, the comparison month is the same calendar month in 2025. March 2026 is compared with March 2025, April 2026 with April 2025, and so on.

    The comparison is made within the same country, market, commodity and unit. Prices are matched by the source-series identifier so that a post-shock observation is compared with the same market-commodity-unit series from the prior year. For each matched month, the country value is the median of (post_price / prior_year_price - 1) x 100 across the matched series.

    The overview keeps the highest positive matched same-month increase per country so that the overview module highlights where prices rose most. Countries with flat, negative or unmatched evidence remain visible in the Country Lens table because those cases matter for interpretation. A positive peak increase means food prices rose in matched WFP/HDX series after the shock window; it does not prove that the escalation caused the increase. Other shocks, seasonality, exchange-rate movements, conflict, market access and assistance changes may occur at the same time.

    Internal/model traceability

    Sensitivity checks are not user-facing scores

    Scenario and weight-sensitivity calculations remain in the workbook as internal audit checks, but they are not presented as separate dashboard results, forecasts or user-facing scores. This keeps the dashboard focused on the official Shock Exposure Index and the raw or near-raw evidence behind it.

    Oil-price scenarios emphasize import exposure, price pressure and assistance pressure. Fertilizer scenarios emphasize agriculture exposure, import exposure and acute food insecurity. Food-price scenarios emphasize domestic price pressure, acute food insecurity and gender vulnerability. Assistance-cut scenarios emphasize assistance pressure, acute food insecurity and displacement. Combined scenarios add the corresponding stress-test assumptions together under low, base and high assumptions. These rows are retained for reproducibility and sensitivity review only.

    Caveats

    Main limitations

    The strongest communication line is to describe exposure, evidence strength and monitoring priority. The dashboard should not be used to claim that the escalation caused a specific food-security outcome, and it should not be used to estimate the number of women or girls who became food insecure. IPC is used as total-population acute food-insecurity severity; public IPC outputs used here are not sex-disaggregated.

    Results should therefore be interpreted as country exposure, evidence strength and monitoring priority across the model evidence categories. Before/after prices describe matched food-price changes after the shock window. Gender vulnerability is a structural and proxy prioritization lens within the official index, while the dashboard shows raw gender indicators directly as cross-country overview evidence and selected-country Country Lens context. The dashboard does not prove causality, estimate women/girls food-security impact, or treat negative, flat or missing price comparisons as evidence that a country has no risk.

    Traceability

    Public sources and calculation trace

    Dashboard values are read from documented public-source inputs and calculation checks. The main trace points are: final country ranking and Shock Exposure Index score; bounded component scores and weights used to calculate that official index; WFP/HDX food-price evidence; latest national IPC timing and severity; IDMC/IOM displacement evidence; OCHA FTS funding evidence; gender/labour/financial-access indicators; and readiness checks for price event-study, food-security outcome and women/girls causal-impact analysis.

    Additional public-source checks provide context and data-gap evidence from ACLED availability, CPI, food incidents, PortWatch and Findex. These layers help explain data availability and context. They do not replace the published Shock Exposure Index.

    Hidden workbook sheets and generated-data traces may still contain scenario sensitivity, weight sensitivity, gender-proxy priority, supplemental-source and other intermediate fields. Those fields are retained for auditability and reproducibility, not as current dashboard outputs.
